# AI-for-Periodontal-Diagnosis
Master’s thesis project focused on applying machine learning models for the classification and prediction of periodontal and apical diseases using clinical and radiographic data, aiming to support diagnostic decision-making in dental practice.

📌 Overview

This repository contains the development of my Master’s thesis, focused on applying machine learning techniques to the classification and prediction of periodontal and apical diseases.

The project integrates clinical and radiographic data to build predictive models that support diagnostic decision-making in dental practice.

🎯 Objectives

Develop and compare machine learning models for disease classification

Identify key risk factors associated with periodontal and apical conditions

Evaluate model performance using standard metrics

Explore the potential of AI as a clinical decision support tool
